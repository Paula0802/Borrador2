Ellipsometry — prototipo

Software para analizar datos de elipsometría espectroscópica: cargar espectros, modelar multilayers con TMM y ajustar parámetros con Gauss–Newton o Levenberg–Marquardt.

¿Qué puede hacer este software?

  • Cargar espectros de Ψ y Δ (CSV / XLSX).
  • Calcular respuesta teórica con TMM para multilayers.
  • Ajustar parámetros (espesores y parámetros de modelos de dispersión).
  • Visualizar n(λ), k(λ), Ψ(λ) y Δ(λ).
